# QuillPlus – Online Assessment Automation (Cypress)

End-to-end test automation suite for an online assessment platform using Cypress.

## What is Covered
- User authentication flows
- Assessment navigation
- Question answering & validations
- Result and feedback workflows

## Tech Stack
- Cypress
- JavaScript
- Cucumber (BDD-style tests)

## Project Structure
- cypress/e2e/features – Feature files
- cypress/e2e/stepdefinitions – Step definitions
- cypress/e2e/tests – Test scenarios
- cypress/support – Custom commands & helpers

## How to Run
1. npm install
2. npx cypress open
